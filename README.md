## Currently working on / En lo que trabajo

- **Image sync pipelines** — bulk image processing & CDN sync for an e-commerce on Shopify (Bun · Next.js middleware · Docker).
- **Internal admin API** — backend para operaciones internas del mismo e-commerce (Bun · TypeScript · Prisma · Redis · Docker).
- **`fe-image-transform-lite`** — open-source macOS app for batch image transforms (Python · Pillow · pywebview).
